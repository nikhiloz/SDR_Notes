# SDR Glossary

> **Comprehensive glossary of Software Defined Radio terminology, abbreviations, and concepts.**

---

## A

| Term | Definition |
|------|------------|
| **ADC** | Analog-to-Digital Converter - Converts analog RF signals to digital samples |
| **ADS-B** | Automatic Dependent Surveillance-Broadcast - Aircraft position broadcasting (1090 MHz) |
| **AFE** | Analog Front End - RF circuitry before digital conversion |
| **AGC** | Automatic Gain Control - Adjusts signal amplitude automatically |
| **AIS** | Automatic Identification System - Maritime vessel tracking |
| **AM** | Amplitude Modulation - Information encoded in signal amplitude |
| **APRS** | Automatic Packet Reporting System - Amateur radio position/data |
| **ASK** | Amplitude Shift Keying - Digital AM modulation |

---

## B

| Term | Definition |
|------|------------|
| **Bandwidth** | Range of frequencies occupied by a signal (Hz) |
| **Baseband** | Signal at original frequencies before modulation |
| **BER** | Bit Error Rate - Measure of digital link quality |
| **BPF** | Band-Pass Filter - Passes frequencies within a range |
| **BPSK** | Binary Phase Shift Keying - 2-state PSK modulation |

---

## C

| Term | Definition |
|------|------------|
| **Carrier** | Unmodulated RF signal used for transmission |
| **CDR** | Clock and Data Recovery - Extracting timing from signal |
| **Channel** | Frequency band allocated for communication |
| **Coherent** | Detection using phase reference |
| **Constellation** | Visual representation of digital modulation states |

---

## D

| Term | Definition |
|------|------------|
| **DAC** | Digital-to-Analog Converter - Converts digital to analog for TX |
| **dB** | Decibel - Logarithmic ratio unit |
| **dBm** | Decibels relative to 1 milliwatt (absolute power) |
| **dBi** | Decibels relative to isotropic antenna (gain) |
| **DC Offset** | Unwanted DC component in I/Q signals |
| **DDC** | Digital Down Converter - Frequency translation in digital domain |
| **Decimation** | Reducing sample rate by factor N |
| **Demodulation** | Extracting information from modulated signal |
| **DMR** | Digital Mobile Radio - Digital voice standard |
| **Doppler** | Frequency shift due to relative motion |
| **DSP** | Digital Signal Processing |
| **DUC** | Digital Up Converter - For transmit applications |
| **Duplex** | Simultaneous two-way communication |
| **Dynamic Range** | Ratio of largest to smallest detectable signal |

---

## E

| Term | Definition |
|------|------------|
| **EIRP** | Effective Isotropic Radiated Power |
| **Envelope** | Amplitude outline of modulated signal |

---

## F

| Term | Definition |
|------|------------|
| **FEC** | Forward Error Correction - Adding redundancy for error recovery |
| **FFT** | Fast Fourier Transform - Efficient spectrum computation |
| **FIR** | Finite Impulse Response - Type of digital filter |
| **FM** | Frequency Modulation - Information encoded in frequency |
| **FPGA** | Field Programmable Gate Array - Reconfigurable hardware |
| **FSK** | Frequency Shift Keying - Digital FM modulation |
| **Full Duplex** | Simultaneous TX and RX on same frequency |

---

## G

| Term | Definition |
|------|------------|
| **Gain** | Signal amplification factor (dB) |
| **GFSK** | Gaussian FSK - Smooth FSK transitions |
| **GNU Radio** | Open-source SDR development framework |

---

## H

| Term | Definition |
|------|------------|
| **Half Duplex** | Alternating TX/RX (not simultaneous) |
| **Harmonic** | Integer multiple of fundamental frequency |
| **HF** | High Frequency - 3-30 MHz |
| **HPF** | High-Pass Filter - Attenuates low frequencies |

---

## I

| Term | Definition |
|------|------------|
| **I/Q** | In-phase/Quadrature - Complex signal representation |
| **IF** | Intermediate Frequency - Between RF and baseband |
| **IIR** | Infinite Impulse Response - Type of digital filter |
| **Image** | Unwanted frequency that mixes to same IF |
| **Impedance** | AC resistance (typically 50Ω or 75Ω) |
| **Interpolation** | Increasing sample rate |
| **ISM** | Industrial, Scientific, Medical - Unlicensed bands |

---

## J-K

| Term | Definition |
|------|------------|
| **Jitter** | Timing uncertainty in sampling |
| **kHz** | Kilohertz - 1,000 Hz |
| **kSPS** | Kilo-samples per second |

---

## L

| Term | Definition |
|------|------------|
| **L-Band** | 1-2 GHz frequency range |
| **LNA** | Low Noise Amplifier - First stage amplification |
| **LO** | Local Oscillator - Tuning reference |
| **LoRa** | Long Range - Low power wide area modulation |
| **LPF** | Low-Pass Filter - Attenuates high frequencies |
| **LSB** | Lower Sideband - SSB below carrier |

---

## M

| Term | Definition |
|------|------------|
| **MIMO** | Multiple Input Multiple Output - Multi-antenna system |
| **Mixer** | Device that combines two frequencies |
| **Modulation** | Encoding information onto carrier |
| **MSPS** | Mega-samples per second |

---

## N

| Term | Definition |
|------|------------|
| **NF** | Noise Figure - Measure of added noise (dB) |
| **NOAA** | National Oceanic and Atmospheric Administration |
| **Notch Filter** | Removes narrow frequency band |
| **Nyquist** | Sampling must be ≥2× highest frequency |

---

## O

| Term | Definition |
|------|------------|
| **OFDM** | Orthogonal Frequency Division Multiplexing |
| **OOK** | On-Off Keying - Simplest digital modulation |
| **Oversampling** | Sampling faster than Nyquist minimum |

---

## P

| Term | Definition |
|------|------------|
| **P25** | Project 25 - Public safety digital radio |
| **Passband** | Frequency range passed by filter |
| **Phase** | Position in signal cycle (degrees/radians) |
| **PLL** | Phase-Locked Loop - Frequency synthesis/tracking |
| **POCSAG** | Paging protocol |
| **PPM** | Parts Per Million - Frequency accuracy |
| **PSK** | Phase Shift Keying - Digital modulation |

---

## Q

| Term | Definition |
|------|------------|
| **QAM** | Quadrature Amplitude Modulation - Combined AM+PM |
| **QPSK** | Quadrature PSK - 4-state PSK (2 bits/symbol) |
| **Quadrature** | 90° phase relationship |

---

## R

| Term | Definition |
|------|------------|
| **RF** | Radio Frequency |
| **RFIC** | RF Integrated Circuit |
| **RX** | Receive/Receiver |
| **RTL-SDR** | Low-cost SDR based on RTL2832U chip |

---

## S

| Term | Definition |
|------|------------|
| **Sample Rate** | Samples per second (SPS) |
| **S-Band** | 2-4 GHz frequency range |
| **SDR** | Software Defined Radio |
| **Selectivity** | Ability to reject adjacent signals |
| **Sensitivity** | Minimum detectable signal level |
| **SMA** | SubMiniature version A - RF connector type |
| **SNR** | Signal-to-Noise Ratio (dB) |
| **Spectrum** | Frequency domain representation |
| **Spurious** | Unwanted signal products |
| **SSB** | Single Sideband - Efficient AM variant |
| **Stopband** | Frequency range rejected by filter |
| **Symbol** | Unit of digital modulation |

---

## T

| Term | Definition |
|------|------------|
| **TCXO** | Temperature Compensated Crystal Oscillator |
| **TETRA** | Terrestrial Trunked Radio - European standard |
| **TX** | Transmit/Transmitter |

---

## U

| Term | Definition |
|------|------------|
| **UHF** | Ultra High Frequency - 300 MHz to 3 GHz |
| **USB** | Upper Sideband - SSB above carrier |
| **USRP** | Universal Software Radio Peripheral (Ettus) |

---

## V

| Term | Definition |
|------|------------|
| **VCO** | Voltage Controlled Oscillator |
| **VHF** | Very High Frequency - 30-300 MHz |
| **VSWR** | Voltage Standing Wave Ratio - Antenna match |

---

## W

| Term | Definition |
|------|------------|
| **Waterfall** | Time-frequency display (spectrogram) |
| **Wideband** | Covering large frequency range |
| **Windowing** | Applying window function to reduce spectral leakage |

---

## X-Z

| Term | Definition |
|------|------------|
| **Zero-IF** | Direct conversion architecture (no IF stage) |
| **Zero-span** | Time-domain view at single frequency |

---
